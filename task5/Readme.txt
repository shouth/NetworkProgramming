# 使用法

**コンパイル**
```
gcc task5.c -lmynet -Imynet -o task5
```

**実行方法**
```
./task5 user_name [port_number]
```
`port_number` はオプショナルであり，指定されなかった場合は `50001` 番を使用します．

# プログラムの特徴

- 処理を関数に分割したのでコードが読みやすく処理が追いやすくなっていると思います．
- サーバーから接続が切られた場合はその旨を表示してクライアントも終了するようにしました．

# 苦労した点

- 文字列処理
    - `strcmp()` が，引数の文字列が等しい場合は `0` を返すのを忘れてそのまま if 文の条件式として利用して条件分岐が上手くいかなかった
    - `fgets()` が改行も含めて返すことを忘れていたので `strcmp()` による照合がうまくいかなかった
- 通信周りの処理で苦労することはなかった
